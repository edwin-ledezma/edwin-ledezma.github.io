Video Narration

The Video above shows the running of my program.In the beggining it shows the first java window that has the live time displaying. I then open up the next window so that i can set the alarm time. Once I set the alarm you can see that the user input is displayed towards the bottom of the screen. Then once the user input and the live clock are at the same time, a dialog box pops up that tells the user that the alarm is going off.

Written Response

In order to make this alarm clock my partner and I encountered quite a few problems. The first problem that we encountered is that we couldn't get a live time to show. We had figured out how to show a timestamp but it wouldn't change time. We then realized that we had to create a while loop so that every second the clock would get replaced with the current time. Another problem that I encountered was that I couldn't get the dialog box to pop up. I realized that the prolem was that I was comparing the user input as a string and not as an integer. Once i changed the user input to be compared as an integer the dialog box functioned as planned.
