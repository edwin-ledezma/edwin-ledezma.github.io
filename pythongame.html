Python Hangman Game

Written Response

For this project my partner and I decided to code a game of Hangman. One difficulty that we encountered was that our code would not print the hangman display when the user had an incorrect guess. We fixed the problem by globally defining 'turns'. Another difficulty that my partner and I encountered was trying to integrate a function into another. We solved the problem by placing the code for calling the second function in a better location in the first function. One part of the code that I completed independently was the graphics displaying the hanging man.
